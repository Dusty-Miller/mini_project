<h2>프로젝트 개요</h2>
1명의 판매자가 다양한 상품을 등록하고, 여러 고객이 이를 구매할 수 있는 중고 거래 쇼핑몰 웹 서비스입니다.<br>
일반 구매 방식뿐만 아니라 낙찰 이벤트(경매 방식)를 도입하여 고객들이 흥미롭게 참여할 수 있도록 구현하였습니다.
